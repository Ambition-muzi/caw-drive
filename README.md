# CawDrive - 无刷电机驱动器



---



# 🚀 简介

* 支持24电压输入
- 当前版本支持SPI，串口和CAN总线通讯，支持TIM PWM刹车功能

- 主控芯片为STM32F446，电机驱动芯片为DRV8323RSRGZR，通过SPI可对驱动芯片完成相关配置

- 支持6xPWM和3xPWM的控制
