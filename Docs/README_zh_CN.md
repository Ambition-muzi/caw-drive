# CawDrive - 无刷电机驱动器

---

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/H2H3PQZVW)

# 🚀 简介

- 支持 24 电压输入

- 当前版本支持 SPI，串口和 CAN 总线通讯，支持 TIM PWM 刹车功能

- 主控芯片为 STM32F446，电机驱动芯片为 DRV8323RSRGZR，通过 SPI 可对驱动芯片完成相关配置

- 支持 6xPWM 和 3xPWM 的控制
